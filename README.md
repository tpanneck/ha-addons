# tpanneck – Home-Assistant-Add-ons

Öffentliches Repository mit eigenen Home-Assistant-Add-ons.

## In Home Assistant hinzufügen

**Einstellungen → Add-ons → Add-on Store → ⋮ (oben rechts) → Repositories**,
dann diese URL eintragen:

    https://github.com/tpanneck/ha-addons

Anschließend erscheinen die Add-ons dieses Repositories im Store.

## Aufbau

Jedes Add-on liegt in einem eigenen Unterordner (mit `config.yaml`, `Dockerfile`, …).
Weitere Add-ons werden hier ergänzt.
